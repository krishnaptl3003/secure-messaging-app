Secure Messaging Application

This project demonstrates a secure client-server messaging system.

Features:
- RSA used for AES session key exchange
- AES used to encrypt message content
- HMAC used for message integrity
- Base64 used for transmission
- Student object serialized to JSON
- Python REST server using Flask
- Python CLI client

How to run:
1. pip install -r requirements.txt
2. python server.py
3. python client.py
